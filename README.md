
# Productivity App

A simple web app to boost productivity with goal tracking, secure data, and tool integrations.

## Features
- **Integration Ecosystem**: Connect your tools in one place.
- **Goal Setting & Tracking**: Define and track goals with achievable tasks.
- **Secure Data Encryption**: End-to-end encryption for data security.

## Components
- **Banner**: Promotional banner.
- **Navbar**: Navigation bar.
- **Hero**: Main hero section.
- **LogoTicker**: Displays partner logos (Acme, Quantum, Echo, Celestial, Pulse, Apex).
- **Features**: Highlights app features.
- **ProductShowcase**: Showcases app interface.
- **FAQs**: Frequently asked questions.
- **CallToAction**: Encourages user action.
- **Footer**: Footer with links/info.

## Setup
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Run the app: `npm start`

## Tech
- React
- TypeScript
- Next.js

## Folder Structure
- `components/`: Reusable components (Banner, Navbar, Hero, etc.)
- `assets/images/`: Logos and app screenshots
- `pages/`: Main page (Home)

## Notes
- Components currently return `null` (placeholders).
- Customize components in `components/` to add UI/functionality.
